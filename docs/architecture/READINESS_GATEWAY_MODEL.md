# Readiness Gateway Model

Karyra Hub exists to make ecosystem exploration safer for local learners.

## Core Principle

```text
Learn → Practice safely → Review readiness → Explore ecosystem
```

Hub is the exploration stage, not the first step.

## Why This Matters

Many blockchain users encounter wallets, signatures, bridges, tokens, and dApps before they understand basic safety. Hub reduces this risk by placing Starknet resources inside a guided context.

## Readiness-Aware Resource Design

Each resource should answer:

1. What is this?
2. Who is it for?
3. What should the learner know first?
4. What risk or confusion should they watch for?
5. What is the next safe step?

## Resource Levels

Suggested levels:

| Level | Meaning |
|---|---|
| Beginner | Safe orientation, concepts, glossary, community introduction |
| Intermediate | Wallet safety, testnet concepts, account abstraction basics |
| Advanced | Cairo, Scarb, Starknet Foundry, Dojo, developer workflows |

## Good Resource Copy

```text
Use this after you understand wallet safety and account abstraction basics.
```

## Risky Resource Copy

```text
Connect your wallet now and start interacting immediately.
```

## Relationship to Passport

The Readiness Passport is the learner's readiness record inside Spark. Hub should respect that idea by not pushing users into high-risk actions before the Passport journey has prepared them.
